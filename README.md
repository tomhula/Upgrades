# Upgrades

**ARCHIVED - DO NOT USE**

A Minecraft SpigotMC plugin written in Java that allows configuring a GUI for players to buy (with in-game currency) certain 'upgrades' that the owner configures as permissions, that unlock a feature of another plugin for the player that bought that upgrade. It also allows leveling upgrades, meaning a player has to first buy the previous level of a certain upgrade before being able to buy the next.
